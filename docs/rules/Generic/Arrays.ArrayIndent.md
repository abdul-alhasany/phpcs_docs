# ArrayIndent

Ensures that array are indented one tab stop.

### Scope

`Generic.Arrays.ArrayIndent`

## Properties

| Property Name | Type | Default | Available Since |
| ------------- | ---- | ------- | --------------- |
| indent        | int  | 4       | 3.2.0           |

## Usage

```xml
<rule ref="Generic.Arrays.ArrayIndent">
    <properties>
        <property name="indent" value="2" />
    </properties>
</rule>
```
