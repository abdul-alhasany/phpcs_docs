# DisallowYodaConditions

Ban the use of Yoda conditions.

### Scope

`Generic.ControlStructures.DisallowYodaConditions`

## Description

Yoda conditions are disallowed.

## Usage

```xml
<rule ref="Generic.ControlStructures.DisallowYodaConditions"></rule>
```
