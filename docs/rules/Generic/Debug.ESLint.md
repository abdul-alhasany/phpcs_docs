# ESLint

Runs eslint on the file.

### Scope

`Generic.Debug.ESLint`

## Properties

| Property Name | Type   | Default | Available Since |
| ------------- | ------ | ------- | --------------- |
| configFile    | string | -       | 2.9.0           |

## Usage

```xml
<rule ref="Generic.Debug.ESLint">
    <properties>
        <property name="configFile" value="/path/to/.eslintrc.json"/>
    </properties>
</rule>
```
