# OneInterfacePerFile

Checks that only one interface is declared per file.

### Scope

`Generic.Files.OneInterfacePerFile`

## Description

There should only be one interface defined in a file.

## Usage

```xml
<rule ref="Generic.Files.OneInterfacePerFile"></rule>
```
