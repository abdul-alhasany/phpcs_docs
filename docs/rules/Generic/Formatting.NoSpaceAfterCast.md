# NoSpaceAfterCast

Ensures there is no space after cast tokens.

### Scope

`Generic.Formatting.NoSpaceAfterCast`

## Description

Spaces are not allowed after casting operators.

## Usage

```xml
<rule ref="Generic.Formatting.NoSpaceAfterCast"></rule>
```
