# TraitNameSuffix

Checks that traits are suffixed by Trait.

### Scope

`Generic.NamingConventions.TraitNameSuffix`

## Description

Trait names must be suffixed with "Trait", e.g. BarTrait.

## Usage

```xml
<rule ref="Generic.NamingConventions.TraitNameSuffix"></rule>
```
