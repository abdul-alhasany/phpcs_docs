# BacktickOperator

Bans the use of the backtick execution operator.

### Scope

`Generic.PHP.BacktickOperator`

## Description

Disallow the use of the backtick operator for execution of shell commands.

## Usage

```xml
<rule ref="Generic.PHP.BacktickOperator"></rule>
```
