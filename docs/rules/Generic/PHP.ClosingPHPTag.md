# ClosingPHPTag

Checks that open PHP tags are paired with closing tags.

### Scope

`Generic.PHP.ClosingPHPTag`

## Description

All opening php tags should have a corresponding closing tag.

## Usage

```xml
<rule ref="Generic.PHP.ClosingPHPTag"></rule>
```
