# DisallowTabIndent

Throws errors if tabs are used for indentation.

### Scope

`Generic.WhiteSpace.DisallowTabIndent`

## Description

Spaces should be used for indentation instead of tabs.

## Usage

```xml
<rule ref="Generic.WhiteSpace.DisallowTabIndent"></rule>
```
