# CamelCapsMethodName

Ensures method names are defined using camel case.

### Scope

`PSR1.Methods.CamelCapsMethodName`

## Description

Method names MUST be declared in camelCase.

## Usage

```xml
<rule ref="PSR1.Methods.CamelCapsMethodName"></rule>
```
