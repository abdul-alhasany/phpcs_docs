# DisallowMultipleStyleDefinitions

Ensure that each style definition is on a line by itself.

### Scope

`Squiz.CSS.DisallowMultipleStyleDefinitions`

## Usage

```xml
<rule ref="Squiz.CSS.DisallowMultipleStyleDefinitions"></rule>
```
