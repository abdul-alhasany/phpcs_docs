# Indentation

Ensures styles are indented 4 spaces.

### Scope

`Squiz.CSS.Indentation`

## Properties

| Property Name | Type | Default | Available Since |
| ------------- | ---- | ------- | --------------- |
| indent        | int  | 4       | 1.4.7           |

## Usage

```xml
<rule ref="Squiz.CSS.Indentation">
    <properties>
        <property name="indent" value="2" />
    </properties>
</rule>
```
