# DisallowSizeFunctionsInLoops

Bans the use of size-based functions in loop conditions.

### Scope

`Squiz.PHP.DisallowSizeFunctionsInLoops`

## Usage

```xml
<rule ref="Squiz.PHP.DisallowSizeFunctionsInLoops"></rule>
```
