# LowercasePHPFunctions

Ensures all calls to inbuilt PHP functions are lowercase.

### Scope

`Squiz.PHP.LowercasePHPFunctions`

## Usage

```xml
<rule ref="Squiz.PHP.LowercasePHPFunctions"></rule>
```
