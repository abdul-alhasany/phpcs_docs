# StaticThisUsage

Checks for usage of $this in static methods, which will cause runtime errors.

### Scope

`Squiz.Scope.StaticThisUsage`

## Description

Static methods should not use $this.

## Usage

```xml
<rule ref="Squiz.Scope.StaticThisUsage"></rule>
```
