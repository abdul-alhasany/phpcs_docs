# CastSpacing

Ensure cast statements don't contain whitespace.

### Scope

`Squiz.WhiteSpace.CastSpacing`

## Description

Casts should not have whitespace inside the parentheses.

## Usage

```xml
<rule ref="Squiz.WhiteSpace.CastSpacing"></rule>
```
