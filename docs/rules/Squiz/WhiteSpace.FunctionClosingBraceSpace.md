# FunctionClosingBraceSpace

Checks that there is one empty line before the closing brace of a function.

### Scope

`Squiz.WhiteSpace.FunctionClosingBraceSpace`

## Usage

```xml
<rule ref="Squiz.WhiteSpace.FunctionClosingBraceSpace"></rule>
```
