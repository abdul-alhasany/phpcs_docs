# FunctionOpeningBraceSpace

Checks that there is no empty line after the opening brace of a function.

### Scope

`Squiz.WhiteSpace.FunctionOpeningBraceSpace`

## Usage

```xml
<rule ref="Squiz.WhiteSpace.FunctionOpeningBraceSpace"></rule>
```
