# PropertyLabelSpacing

Ensures that a property or label colon has a single space after it and no space before it.

### Scope

`Squiz.WhiteSpace.PropertyLabelSpacing`

## Usage

```xml
<rule ref="Squiz.WhiteSpace.PropertyLabelSpacing"></rule>
```
