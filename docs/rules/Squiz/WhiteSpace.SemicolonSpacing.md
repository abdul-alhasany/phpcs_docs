# SemicolonSpacing

Ensure there is no whitespace before a semicolon.

### Scope

`Squiz.WhiteSpace.SemicolonSpacing`

## Description

Semicolons should not have spaces before them.

## Usage

```xml
<rule ref="Squiz.WhiteSpace.SemicolonSpacing"></rule>
```
